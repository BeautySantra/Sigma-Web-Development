CSS Selector

In CSS, selectors are used to target HTML elements and apply styles to them.

  1. Type Selector (Element Selector): Targets elements by their tag name
  2. Class Selector: Targets elements by their class attribute
  3. ID Selector: Targets elements by their id attribute
  4. Universal Selector: Targets all elements on the page
  5. Attribute Selector: Targets elements based on their attributes
  6. Grouping Selectors: Allows you to apply the same styles to multiple selectors
  7. Pseudo-class Selector: Targets elements based on their state or position
  8. Pseudo-element Selector: Targets specific parts of an element
  9. Descendant Selector: Targets elements that are descendants of a specified element
  10. Child Selector: Targets elements that are direct children of a specified element
  11. Adjacent Sibling Selector: Targets elements that are immediately preceded by a specified element
  12. General Sibling Selector: Targets elements that are siblings of a specified element
